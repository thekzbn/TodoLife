Get Started in Minutes

Simple setup process to begin your spiritual productivity journey

1

Clone Repository

Download TodoLife from GitHub to get started

git clone https://github.com/thekzbn/todolife.git
2

Setup Firebase

Configure authentication and cloud sync

  • Create a Firebase project
  • Enable Authentication and Firestore
  • Copy config to firebase-config.js
3

Run Locally

Start using TodoLife on your local machine

python -m http.server 8000
